TrovaLaSondaFw

What's this?

A firmware for the Heltec Wifi LoRa32 V3 and LilyGo TTGO LoRa32 V2.1_1.6 boards (433MHz version only). Its purpose is receiving meteorological radiosondes. To be used in conjunction with your phone and the Trova La Sonda app (beta) for Android. Be sure to select "Bluetooth Low Energy" to connect to the device.

Is it free?

Absolutely. A donation to a charity would be much appreciated as a way to say thanks. I especially recommend the Landmine Relief Fund.

Is it open source?

Definitely. See the code on github.

Did you do all this?

We all stand on the shoulders of giants. Part of the decoding code is taken from sondedump, the RS decoders and the rdz_ttgo_sonde firmware. This page was made possible by esp-web-tools.

Is it stable?

As of today (November 30, 2024) it has undergone little testing, so beware if you rely solely on this solution to locate sondes in the wild.

Which kind of radiosondes can it receive/decode?
